# Juego-Truco-JS-HTML-CSS
Este proyecto personal, recrea el tradicional juego de cartas argentino Truco usando HTML, CSS y JavaScript. La aplicación simula una partida interactiva con las reglas clásicas, incluyendo jugadas como el envido y el truco, todo en una interfaz visual atractiva y adaptable. Con JavaScript para manejar la lógica del juego y CSS para animaciones y diseño, este proyecto busca capturar la emoción y la estrategia de uno de los juegos más emblemáticos de Argentina
